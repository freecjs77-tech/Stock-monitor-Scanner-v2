# 실행 규칙 — 기술적 차트분석 전용 시스템

## 종목 분석 요청을 받으면

1. `agents/00_오케스트레이터.md`를 읽고 전체 흐름을 따른다
2. Phase 1~5를 순서대로 실행한다
3. 기술적 분석은 `agents/05_기술적분석.md`의 지시를 따른다
4. 모든 분석에서 **반드시 웹 검색으로 실제 데이터를 확인**한다
5. 결과를 `agents/06_리포트생성.md` 양식에 따라 PDF로 `reports/` 폴더에 저장한다
6. 채팅으로 결과 요약을 보고한다

## 데이터 원칙

- 웹 검색으로 확인된 실제 수치만 사용한다
- 검증되지 않은 수치를 임의로 만들지 않는다
- 찾을 수 없는 데이터는 "데이터 미확인"으로 표기한다

## 분석 범위

**본 시스템은 기술적(차트) 분석만 수행한다.**
- ✅ 이동평균선, RSI, MACD, 스토캐스틱
- ✅ 볼린저밴드, ATR (변동성)
- ✅ 거래량 분석, OBV
- ✅ 차트 패턴 (헤드앤숄더, 더블탑/바텀 등)
- ✅ 지지선/저항선, 피보나치
- ✅ 매수/매도 신호 판정 및 매매 전략
- ❌ 재무분석 (PER, ROE, 매출 등) — 수행하지 않음
- ❌ 산업분석 — 수행하지 않음
- ❌ 모멘텀/뉴스 분석 — 수행하지 않음
- ❌ 리스크분석 — 수행하지 않음

## 보고 형식

분석 완료 후 채팅으로 알려줄 내용:
- 종목명 (티커)
- 기술적 의견 + 종합점수 (/85)
- 매수/매도 신호 핵심 3가지
- 1차 목표가 + 손절 기준
- PDF 파일 위치
