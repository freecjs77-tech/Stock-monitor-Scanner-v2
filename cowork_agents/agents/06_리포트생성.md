### 2페이지: 세부 분석

```
A. 이동평균선 분석   — 신호 테이블 (정배열/역배열, 이격도)
B. 모멘텀 지표       — RSI/MACD/스토캐스틱 수치 + 신호
C. 볼린저밴드        — 상/중/하단 + %B + 밴드폭 상태
D. 거래량 분석       — 현재/평균 비교 + OBV 방향
E. 차트 패턴         — 인식된 패턴 + 지지/저항선 테이블
F. 매매 전략         — 진입 조건, 목표가, 손절 기준, R/R 비율
```

```

예) 삼성전자_005930_차트분석_20260321.pdf

```

저장 위치: `reports/` 폴더

## 디자인 원칙

**색상 기준:**
- 강한 매수/매수 신호: 녹색 (#1A7A3C)
- 중립 신호: 주황 (#D35400)
- 매도/강한 매도 신호: 빨강 (#C0392B)
- 매수 셀 배경: 연한 녹색 (#D4EDDA)
- 매도 셀 배경: 연한 빨강 (#FDECEA)

**레이아웃:**
- 각 분석 섹션에 점수 표시 (예: `A. 추세 분석  14/20점`)
- 매수/매도 신호는 색상으로 즉시 구분
- 지지선/저항선 테이블 필수 포함
- 매매 전략 박스 (진입/목표/손절/R:R)
- 면책 조항 하단 소형 글씨

**스코어 바:**
점수를 시각적 막대로 표현:
- 사용 가능한 경우 컬러 셀(테이블)로 표현
- 예: 14/20점 → 녹색 셀 14칸 + 회색 셀 6칸

## 코딩 지침

`HYGothic-Medium` CID 폰트를 반드시 사용한다:
```python
from reportlab.pdfbase import pdfmetrics
from reportlab.pdfbase.cidfonts import UnicodeCIDFont
pdfmetrics.registerFont(UnicodeCIDFont('HYGothic-Medium'))
FONT = 'HYGothic-Medium'
```

점수 바 구현 (테이블 방식):
```python
def score_bar(score, max_score, filled_color, empty_color=colors.HexColor("#E0E0E0")):
    """점수를 컬러 블록 바로 시각화"""
    cells = []
    for i in range(max_score):
        bg = filled_color if i < score else empty_color
        cells.append(Paragraph(" ", ParagraphStyle('bar', fontName=FONT, fontSize=6)))
    t = Table([cells], colWidths=[3.5*mm]*max_score, rowHeights=[5*mm])
    t.setStyle(TableStyle([
        ('BACKGROUND', (0,0), (score-1, 0), filled_color),
        ('BACKGROUND', (score,0), (-1,0), empty_color),
        ('GRID', (0,0), (-1,-1), 0.3, colors.white),
    ]))
    return t
```
